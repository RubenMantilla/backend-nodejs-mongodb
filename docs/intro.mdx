Bienvenido a este pequeño curso donde aprenderemos conceptos fundamentales en el desarrollo web orientado al backend, utilizando NodeJS y MongoDB.

Este curso está dirigido a personas que deseen adentrarse en el mundo del desarrollo web, y que deseen aprender a crear aplicaciones del lado del servidor desde cero.

El curso consta de ocho módulos, los cuales se dividen en pequeñas partes, con el objetivo de que puedas aprender de forma gradual y sencilla.

- [Módulo 1: Introducción a Javascript](/category/modulo-01)
- [Módulo 2: Conceptos avanzados de JS](/category/modulo-02)
- [Módulo 3: Introducción a NodeJS](/category/modulo-03)
- [Módulo 4: Introducción a Express](/category/modulo-04)
- [Módulo 5: Gestión de peticiones HTTP](/category/modulo-05)
- [Módulo 6: Uso de Middlewares en Express](/category/modulo-06)
- [Módulo 7: Introducción a MongoDB](/category/modulo-07)
- [Módulo 8: Uso de bases de datos con NodeJS y MongoDB](/category/modulo-08)

Como proyecto final de este curso desarrollaremos una pequeña API Rest con NodeJs y MongoDB, la cual nos permitirá gestionar una lista de tareas, que si bien parece algo sencillo, nos permitirá aprender conceptos importantes a la hora de desarrollar cualquier tipo de API Rest.

El desarrollo del proyecto será en etapas, de forma que podamos ir viendo los avances de la misma, y que podamos ir aprendiendo de forma gradual. Del mismo modo, podrás seguir el paso a paso desde [aquí](/category/final-project).
